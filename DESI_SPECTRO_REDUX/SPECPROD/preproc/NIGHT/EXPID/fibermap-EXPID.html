fibermap-EXPID.fits¶

Summary

fibermap augmenting the input fiber assignment file with information about where the fibers actually ended up from the coordinates file.

Naming Convention

fibermap-{expid}.fits, where {expid} is the zero-padded 8-digit exposure ID.

Regex
