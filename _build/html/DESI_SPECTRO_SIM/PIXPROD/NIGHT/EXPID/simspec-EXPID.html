simspec-EXPID.fits

Summary:

Input spectra to simulate with pixsim.

Naming Convention:

simspec-{EXPID}.fits, where {EXPID} is the 8-digit exposure ID.

Regex:

simspec-[0-9]{8}\.fits

File Type:

FITS, 2 GB
