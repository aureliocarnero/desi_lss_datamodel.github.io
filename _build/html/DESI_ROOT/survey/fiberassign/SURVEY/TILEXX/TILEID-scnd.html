TILEID-scnd.fits

Summary:

This file contains the secondary targets covered by the tile disk-footprint.

Naming Convention:

{TILEID}-scnd.fits, where {TILEID} is the zero-padded, 6-digit TILED.

Regex:

[0-9]{6}-scnd\.fits

File Type:

FITS, 1 MB

Contents

EXTNAME

Type

BINTABLE

Secondary targets covered by the tile disk-footprint.

FITS Header Units

HDU0

This HDU has no non-standard required keywords.

Empty HDU.

HDU1

EXTNAME = TARGETS

Secondary targets covered by the tile disk-footprint: those are read from the MTL ledgers and desitarget catalogs and provided as input to fiberassign.

Notes and Examples

For the SURVEY=cmx m33 tile (TILEID=80615) tile and all the SURVEY=sv1 tiles (except TILEID=80971-80976, the dc3r2 ones), proper-motion correction was applied at the fiberassign design step; thus the following columns can have different values than in the desitarget products: TARGET_RA, TARGET_DEC, REF_EPOCH, PLATE_RA, PLATE_DEC, and PLATE_REF_EPOCH.
