mtl-dark-hp-HPX¶

Summary

MTL ledgers used to override information for DESI dark-time secondary targets.

Naming Convention

mtl-override-dark-hp-HPX.ecsv, where HPX is the nside=32 (NESTED) HEALPixel integer.

Regex

mtl-override-dark-hp-[0-12287]\.ecsv

File Type

ecsv, 0-10 KB
