DESI_SURVEYOPS contains data files used for day-to-day survey operations, with the canonical location of $DESI_ROOT/survey/ops/surveyops/trunk/, though data releases will use a tagged version. The main directory is an SVN trunk that is updated as the DESI survey proceeds.

Subdirectories:

  • mtl
  • ops
