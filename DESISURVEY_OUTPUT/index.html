DESISURVEY_OUTPUTÂ¶

DESISURVEY_OUTPUT contains the outputs of desisurvey and surveysim, and is expected to be manually by users of these packages. There is no one, official location for these files in the DESI directory tree.

We expect this directory structure will be updated as part of integrating desisurvey (afternoon planning and next tile selector) into online operations.

Warning

These files are not expected to be released, and DESISURVEY_OUTPUT is not normally defined automatically in the DESI environment.

  • ephem_{start}-{stop}.fits : cached ephemeris for survey planning
  • surveyinit.fits : Design hour angles?
  • planner_YEAR-MM-DD.fits : Afternoon planner state
  • scheduler_YEAR-MM-DD.fits : Next tile selector state
  • stats_surveysim.fits : per-tile and per-night statistics from surveysim run
  • exposures_surveysim.fits : exposure metadata from surveysim run
