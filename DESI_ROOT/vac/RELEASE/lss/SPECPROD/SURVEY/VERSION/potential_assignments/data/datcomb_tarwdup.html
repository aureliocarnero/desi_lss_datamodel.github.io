datcomb_tarwdup¶

Summary:

File with information about all targets after match with potential assigment from FA.

Naming Convention:

datcomb_{PROGRAM}_tarwdup_alltiles.fits, where {PROGRAM} refers to the observing program, either dark or bright.

Regex:

datcomb_[a-z]{4,6}_tarwdup_alltiles.fits

File Type:

FITS, 1 GB
