Welcome to the DESI Data Model!

The DESI Data Tree

These pages define the directory structure and file formats for DESI data products relative to a root directory DESI_ROOT. Each data release has its own DESI_ROOT, e.g. the Data Release 1 at https://data.desi.lbl.gov/public/dr1.

Data Directories

Directories under DESI_ROOT:

  • spectro/redux/SPECPROD/: Processed spectra, classifications, and redshifts

  • survey/catalogs/RELEASE: DR Large Scale Structure catalogs and associated mock catalogs

  • vac/RELEASE/: Value Added Catalogs

  • target/: Target selection and fiber assignment catalogs

  • spectro/data/NIGHT/EXPID/: Raw data

The following directories are more expert-level (e.g. pipeline calibration inputs) and are documented for DESI collaboration internal use and may not be included in data releases:

  • survey/ops/surveyops/: Data files used for day-to-day survey operations

  • spectro/desi_spectro_calib/: Spectrograph calibration data

  • protodesi/: Data and logs from the ProtoDESI campaign (no spectra)

  • $DESISURVEY_OUTPUT: Outputs from desisurvey and surveysim

  • $DESI_SPECTRO_SIM: Simulated spectro data

  • $DESIMODEL: Data used for simulating DESI
