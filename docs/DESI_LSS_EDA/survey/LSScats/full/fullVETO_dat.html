The “full” LSS catalogs for data¶

Summary

LSS catalogs containing information on all targets identified as reachable by DESI fiberassign, with one entry for each. The files are split by target type and whether of not vetos for angular positions have been applied.

Naming Convention

{TARGET}_full{VETO}.dat.fits, where {TARGET} is the target type: QSO, ELG, ELGnotqso, ELG_HIP, ELG_HIPnotqso, LRG, LRG_main, for dark or BGS_ANY, BGS_BRIGHT for bright. {VETO} is _noveto if vetos have not been applied and blank otherwise

Regex

[a-zA-Z]{3,}_full[a-z]{0,7}.dat.fits

File Type

FITS, 1 GB (size varies considerably depending on target type)
