qso_mgii-SURVEY-PROGRAM-PIXNUM.fitsÔÉÅ

Summary:

This file contains the output of the MgII fitter which is a classifier algorithm to collect spectra with MgII broad emission line.

Naming Convention:

qso_mgii-SURVEY-PROGRAM-PIXNUM.fits, where SURVEY is e.g. main or sv1, PROGRAM is e.g. bright or ``dark and PIXNUM is the HEALPixel number.

Regex:

qso_mgii-(cmx|main|special|sv1|sv2|sv3)-(backup|bright|dark|other)-[0-9]+\.fits

File Type:

FITS, 22 KB
