SPECPROD

Default $DESI_ROOT/spectro/redux/SPECPROD

DESI spectroscopic data processing productions are grouped by spectroscopic production names (SPECPROD), named after mountains for data releases, e.g. “iron” for Data Release 1.

Files in the top-level production directory:

  • exposures-SPECPROD.fits: summary of exposures

  • tiles-SPECPROD.fits: summary of tiles

Subdirectories under a spectroscopic production:

  • zcatalog/: summary redshift catalogs

  • healpix/SURVEY/PROGRAM/PIXGROUP/PIXNUM: spectra, classifications, and redshifts coadded across tiles, grouped by survey, program, and healpixel.

  • tiles/GROUPTYPE/TILEID/GROUPID/index: spectra, classifications, and redshifts coadded per tile.

  • calibnight/NIGHT/: nightly calibrations

  • preproc/NIGHT/EXPID: preprocessed spectrograph CCD images

  • exposures/NIGHT/EXPID: per-exposure intermediate files

Other subdirectories that represent internal desispec pipeline details, but do not contain production data:

  • exposure_tables/ : Details on individual exposures that are inputs to the pipeline. For example, bad exposures may be marked in these tables.

  • processing_tables/ : Details on the processing status of individual exposures.

  • run/ : Scripts and log files produced by the desispec pipeline.
