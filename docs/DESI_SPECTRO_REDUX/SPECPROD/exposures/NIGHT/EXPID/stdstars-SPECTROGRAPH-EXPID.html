stdstars-SPECTROGRAPH-EXPID.fits

Summary:

This file contains the normalized standard star models fitted to the frame data.

Naming Convention:

stdstars-{SPECTROGRAPH}-{EXPID}.fits where {SPECTROGRAPH} is the single-digit spectrograph number 0-9, and {EXPID} is the zero-padded 8-digit exposure number.

Regex:

stdstars-[0-9]-[0-9]{8}\.fits

File Type:

FITS, 5 MB

FITS Header Units

HDU0

EXTNAME = FLUX

2D array of best fit standard star model fluxes, in units of 10^{-17} ergs/s/cm2/A, at high resolution, including the Milky Way extinction, and the Doppler shift from the fitted radial velocity. The size of the array is [nspec, nwave]. nspec is the number of standard stars and nwave in the length of the wavelength array.
