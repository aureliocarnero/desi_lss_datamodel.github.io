on_sky_movedata

Summary:

Contains a record of positioner move commands and results.

Naming Convention:

{posname}_{seqid}_on_sky_movedata.csv

Regex:

(UM[0-9]{5})(_)([0-9]{4})(-)([0-9]{2})(-)([0-9]{2})(_)(T)([0-9]{6})(_on_sky_movedata.csv)

File Type:

csv

Warning

This description does not correspond to any file in the actual ProtoDESI logs.
