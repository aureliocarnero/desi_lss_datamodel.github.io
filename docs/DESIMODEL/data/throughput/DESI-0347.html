DESI-0347

Summary:

Focal plane distortion maps. ECSV files may also be present.

Naming Convention:

DESI-0347_random_offset_{N}.fits, where {N} is the (arbitrary) number of the random realization.

Regex:

DESI-0347_random_offset_[0-9]+\.fits

File Type:

FITS, 525 KB
