Vfilt¶

Summary:

V-band filter response function.

Naming Convention:

Vfilt.txt

Regex:

Vfilt\.txt

File Type:

ASCII, 4.5 KB

Contents¶

This ASCII file contains two columns, wavelength [Å] and filter response [fraction transmitted].
