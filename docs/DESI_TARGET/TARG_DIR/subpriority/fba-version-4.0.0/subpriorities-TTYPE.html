supplemental skies¶

Summary

The subpriority files are single binary tables that include the fixed values of subpriority enforced for Main Survey targets. There is one file for each of dark-time and bright-time targets, and one for blank sky locations.

Naming Convention

subpriorities-TTYPE.fits, where TTYPE is one of dark, bright or sky.

Regex

subpriorities-[a-zA-Z]+\.fits

File Type

FITS, 41 - 360 MB
