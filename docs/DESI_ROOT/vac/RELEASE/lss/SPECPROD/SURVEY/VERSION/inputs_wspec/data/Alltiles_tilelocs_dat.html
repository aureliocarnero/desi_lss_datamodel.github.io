Alltiles_tilelocs¶

Summary:

Information on the tiles and locations each target appears on.

Naming Convention:

Alltiles_{PROGRAM}_tilelocs.dat.fits, where {PROGRAM} denotes the observing program, either dark or bright.

Regex:

Alltiles_[a-z]{4,6}_tilelocs.dat.fits

File Type:

FITS, 544 MB
