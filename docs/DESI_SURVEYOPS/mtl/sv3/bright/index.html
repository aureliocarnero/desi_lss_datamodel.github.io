brightÔÉÅ

The bright directory hosts MTL ledgers for the DESI bright-time program. Targets are stored in ledgers that are split by HEALPixel (in the NESTED scheme) at a resolution of nside=32.

The filename for each ledger resembles sv3mtl-bright-hp-HPX.ecsv, where HPX is the healpixel number.

  • sv3mtl-bright-hp-HPX.ecsv : MTL ledgers for the DESI bright-time program
