ToO¶

Summary:

Targets of Opportunity ledger

Naming Convention:

ToO.ecsv

Regex:

ToO\.ecsv

File Type:

ecsv, 112 KB

Notes¶

The unit nanomaggy in this file is actually recorded as nmgy. This unit will be read correctly into an astropy table from a .ecsv file.
